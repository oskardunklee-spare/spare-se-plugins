# Scope of Spare

**This file is not an authoritative scope statement.** It is a product-catalog index. Use it to know which product names to search for in past RFPs and Spare docs when answering a matrix. Do not use it to conclude that Spare does or does not support a specific feature; that determination always comes from live evidence per Rule 0 in `methodology-rules.md`.

Spare's product catalog is broad and updates frequently. Any SE reading an incoming RFP should assume that some subset of its requirements maps to Spare products and search the live sources accordingly.

## Why this file is minimal

Earlier versions of this file described what Spare does and does not ship. Those descriptions aged poorly; the product updates faster than a reference file can be maintained, and the cached claims caused wrong outputs (including false "this is not a Spare deal" declarations on RFPs Spare can actually answer).

The methodology now requires that scope questions be answered by searching the live sources per row. This file exists only to scaffold that search by listing the names to look for.

## Contributing to this file

When a new product or major module ships, add its canonical name here so the skill can search for it. Do not add feature-level claims (e.g., "supports barcode scanning") here; those belong in the sourced response for a specific RFP row, cited from a past RFP or a doc page.
